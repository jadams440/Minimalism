Game Controls

← → : Move
↑ : Jump
↓ : Swap color
space : Change player
r : Reset level

Menu Controls

↑ ↓ ← → : Move cursor
space : start level

Rules

• Guide each player to the corresponding exit
• Each player only reacts to their own color (and black)
• The color behind the current player can only be changed if neither of the others are standing over that block of color

Challenges

• Change the colors the fewest times
• Leave the fewest colors remaining
Canvas not supported!
